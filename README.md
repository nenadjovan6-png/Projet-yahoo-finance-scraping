# Outil automatisé de Web Scraping - Yahoo Finance

## Description
Ce projet extrait automatiquement les 200 actions les plus actives de Yahoo Finance. Il convertit les données financières en format numérique et les sauvegarde en CSV.

## Fonctionnalités
- **Automatisation** : Script exécutable en une ligne.
- **Éthique** : Respect du robots.txt et délais entre les requêtes (3s).
- **Robustesse** : Gestion des erreurs de connexion et des balises HTML manquantes.

## Installation
1. Cloner le projet : `git clone https://github.com/TON_NOM/yahoo-finance-scraper.git`
2. Installer les dépendances : `pip install -r requirements.txt`

## Utilisation
Lancer le script avec : `python scraper.py`

## Mentions Légales
Ce projet est réalisé dans un but pédagogique. L'utilisation des données doit respecter les conditions d'utilisation de Yahoo Finance.
